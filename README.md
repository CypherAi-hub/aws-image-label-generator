# AWS AI Image Label Generator 🧠☁️

## Overview
This project uses AWS Rekognition and S3 to analyze images and automatically generate labels.

## Features
- Upload images to S3
- Detect objects using AWS Rekognition
- Generate labels automatically
- Python-based processing

## Tech Stack
- AWS S3
- AWS Rekognition
- Python (boto3)

## Why I Built This
I built this project to learn how AI services integrate with cloud infrastructure and to demonstrate real-world use of AWS.
